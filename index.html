visualisation by Matthew Coller, for Founders and Survivors

This is a preliminary visualisation, to show how the visualisation will work in a browser. It currently requires the Google Earth plug-in (installed automatically when you install Google Earth). However, the underlying data is in standard KML, so the visualisation platform could be changed in future.

Notes:

  • The symbology used here is preliminary, and may be somewhat inappropriate - open for discussion!
  • This visualisation is made using simple lat-lon values, so multiple instances of the same place appear as icons piled on top of one another. Each icon is semi-transparent, so clusters show as more opaque sets. The geocoding also includes an 'accuracy' attribute that would allow randomisation of location, or possible density temporal mapping.
